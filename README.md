# Kotlin Homepage - React Router 7 Migration

A migration of the Kotlin website homepage to React Router 7 Framework Mode with Server-Side Rendering (SSR).

## Tech Stack

- **React Router 7**
- **React 19**
- **TypeScript**
- **SCSS** 
- **@rescui** 
- **Vite**

## Prerequisites

- **Node.js** v22.11.0 or higher

## Running locally
```bash
npm install
npm run dev
```

The site will be available at [http://localhost:5173](http://localhost:5173).
